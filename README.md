# Netshoes Exercício 3

Este projeto implementa a solução para o exercício 3 do teste para o Netshoes.

Dada uma stream, é retornado o primeiro caractere que não se repete em toda a stream. O método que realiza esta operação é o método de nome firstChar encontrado na classe FirstChar.java

A implementação foi criada tendo em mente a performance da operação.

##### Classes:

FirstChar.java:  
contém o método que realiza a operação principal contida na descrição do exercício

Stream.java:  
interface da stream

StreamImpl.java:  
uma implementação da stream

Main.java:  
possui o método main para execução do código  

##### Testes:

FirstCharTest.java:  
testa a operação principal
